Convertir un nombre en tableau inversé de chiffres
Étant donné un nombre aléatoire non négatif, vous devez renvoyer les chiffres de ce nombre dans un tableau dans l'ordre inverse.

~~~js
function digitize(n) {
  //code here
}
~~~

Exemple (Entrée => Sortie) :
~~~js
35231 => [1,3,2,5,3]
0 => [0]
~~~
